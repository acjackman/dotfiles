# Dotfiles

These are my dotfiles, massively easier system configuration.

Configured with [Chezmoi](https://www.chezmoi.io/).

## Updating dotfiles

```sh
chezmoi update
```


## Bootstrapping

```sh
/bin/sh -c "$(curl -fsSL https://raw.githubusercontent.com/acjackman/dotfiles/main/bootstrap.sh)"
```
