# Shell Conventions

## Shebangs

- `#!/usr/bin/env zsh` or `#!/usr/bin/env bash` for portability
- `#!/bin/sh` for POSIX-compliant scripts

## Style

- Prefer `[[ ]]` over `[ ]` in bash/zsh
- Minimal comments — only when logic isn't self-evident
- Zsh is the default shell; zsh-specific features are acceptable
